setwd("D:/R/DUA/Watermark/")

Memanggil Package

library(EBImage)

Sumber Gambar

Image <- readImage('UIN-Malang.jpg')

Menampilkan Gambar

display(Image)

Membuat Watermark

Menginstall Package Magick

Untuk membuat watermark, kita perlu menginstall package magick terlebih dahulu. dengan kode

install.packages(‘magick’)

Setelah melakukan instalasi, kita perlu memanggil packagenya agar bisa digunakan. 

library(magick)

Watermark tulisan pada gambar.

Disini adalah cara menambahkan watermark berupa tulisan pada gambar tertentu. 

magick::image_annotate(img, "Logo of UIN MALANG", size = 50, gravity = "northwest", location = "+120+627",degrees = 0, color = "transparent", strokecolor = "#FF2D00", boxcolor = NULL)

Watermark tulisan pada website.

Untuk menambahkan watermark pada website kita bisa menuliskan seperti ini.

Titik-titik dibawah bisa kita ubah sesuai dengan tulisan yang kita inginkan.

## [1] "<div class=\"watermark\">...</div>"

Watermark gambar pada website.

Untuk menambahkan watermark pada website kita bisa menuliskan seperti ini.

Titik-titik dibawah bisa kita ubah sesuai dengan nama file gambar yang ada di lokasi kerja.

## [1] "<div class=\"watermark\"><img src=\"...\"></div>"
